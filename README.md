
# Система автоматического составления расписания 📅

Комплексное решение для автоматизации учебного процесса, включающее в себя ядро генерации расписания, веб-интерфейс, систему администрирования и Telegram-бота.

## 🚀 Функциональные модули

### 1. Веб-интерфейс (Сайт)
* Просмотр расписания на семестр с фильтрацией по группам, преподавателям и кабинетам.
* Генерация отчетов по расписанию.

### 2. Telegram-бот
* Оперативные уведомления о начале пар и изменениях в расписании.
* Рассылка новостей филиала.
* Личный кабинет пользователя для настройки уведомлений.

### 3. Панель администратора
* Модули внесения учебной нагрузки и управления доступностью аудиторий.
* Формы регистрации и ввода данных для преподавателей.
* Интерфейс для запуска алгоритма, корректировки и публикации финального расписания.

### 4. Алгоритм генерации
* Автоматизированный расчет сетки занятий с учетом заданных ограничений.

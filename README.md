## 🚀 О проекте

**GoShiet** — это мощное решение для автоматизации образовательного процесса. Проект объединяет в себе умное ядро генерации расписания, удобный веб-интерфейс, систему администрирования и Telegram-бота для мгновенных уведомлений. 

---

## 🎭 Ролевая модель системы

Система поддерживает четкое разделение прав доступа для разных типов пользователей:

### 👑 КОРОЛИ (Суперпользователи)
* Глобальное управление: создание структуры университета и профилей преподавателей.

### 🛡️ Администраторы
* Полный контроль над преподавательским составом (добавление через ссылку).
* Управление инфраструктурой: создание направлений, курсов и аудиторий.
* Наполнение базы: заполнение данных о предметах (название, выбор преподавателя, сложность в баллах, количество часов в семестре).
* Управление расписанием: запуск алгоритма генерации, ручная корректировка и публикация.
* Коммуникация: подтверждение и рассылка уведомлений об изменениях.

### 🎓 Преподаватели
* Регистрация: заполнение анкеты (ФИО, ученое звание, контакты).
* Гибкая настройка графика: указание желаемого количества пар подряд, пар в день и доступных дней для недельного расписания.

### 👀 Студенты (Гости)
* Свободный доступ: использование сайта в режиме гостя без необходимости регистрации.

## 🧩 Функциональные модули

### 🌐 1. Веб-интерфейс (Сайт)
* **Умные фильтры:** поиск расписания по университету, направлению и курсу, а также точечные фильтры "под студента", "под преподавателя", "под аудиторию".
* **Визуализация:** красивая и интуитивно понятная недельная сетка расписания.
* **Машина времени:** удобная смена даты для просмотра предыдущего и будущего (приблизительного) расписания.
* **Дашборд:** стильный блок с численной аналитикой, расположенный прямо под сеткой расписания.

### 🤖 2. Telegram-бот
* Оперативные push-уведомления о начале пар и любых изменениях в расписании.
* Рассылка важных новостей филиала.
* Личный кабинет для гибкой настройки подписок.

### ⚙️ 3. Алгоритм генерации
* Автоматизированный математический расчет сетки занятий, строго учитывающий все заданные ограничения преподавателей и аудиторий.

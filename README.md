## О проекте

**GoShiet** — комплексное решение для автоматизации образовательного процесса. Проект объединяет умное ядро генерации расписания, веб-интерфейс, систему администрирования и Telegram-бота для мгновенных уведомлений.

## 🧩 Функциональные модули

### 🌐 `MODULE_01` — Веб-интерфейс

- **Умные фильтры** — по университету, направлению, курсу; точечные фильтры под студента, преподавателя и аудиторию
- **Визуализация** — интуитивно понятная недельная сетка расписания
- **Машина времени** — навигация по прошедшим и будущим датам
- **Дашборд** — блок с численной аналитикой под сеткой расписания

### 🤖 `MODULE_02` — Telegram-бот

- **Push-уведомления** о начале пар и любых изменениях в расписании
- **Рассылка** важных новостей и объявлений филиала
- **Личный кабинет** — гибкая настройка подписок

### ⚙️ `MODULE_03` — Алгоритм генерации

- **Математический расчёт** сетки занятий с учётом всех заданных ограничений
- **Ограничения преподавателей** — желаемые пары подряд, пары в день, доступные дни
- **Ограничения аудиторий** — автоматическое распределение по доступным помещениям
- **Ручная корректировка** — возможность дополнительного редактирования после генерации
